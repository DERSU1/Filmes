# Cinema Production

Um catálogo interativo de filmes que apresenta sinopses e resenhas das principais produções cinematográficas do momento.

### Demonstração
Acesse o projeto online: (https://filmes-mauve-theta.vercel.app)

### Funcionalidades
- Galeria de filmes atualizada (ex: Bad Boys, Divertida Mente 2, Meu Malvado Favorito 4).
- Seção "Sobre" detalhando o objetivo do site.
- Links diretos para redes sociais e integração visual com a API TMDB.

### Tecnologias Utilizadas
- **Frontend:** HTML5, SCSS para estilização avançada e JavaScript para interatividade.
- **Deploy:** Vercel.
